# 004 PRD

## Feature 1. 주문 생성
- 사용자는 메뉴 이름, 수량, 가격을 입력할 수 있다.
- 주문 생성 시 유효하지 않은 값은 거부한다.

## Feature 2. 주문 목록 조회
- 현재까지 생성된 모든 주문을 목록으로 확인할 수 있다.

## Feature 3. 총 금액 계산
- 모든 주문의 (가격 × 수량)을 합산하여 총 금액을 보여준다.

## Validation Rules
- 메뉴 이름은 비어 있으면 안 된다.
- 수량은 1 이상이어야 한다.
- 가격은 0보다 커야 한다.

## Output Example
- 주문 생성 성공
- 주문 3건 조회
- 총 금액: 28000원
